# LEGACY Pro Wrestling 8.5.5 — Championship Path

- World Champion record appears in the Power Rankings champion tile.
- Champion is excluded from ordinary random matches and world simulation.
- Champion cannot lose unless a World Championship match changes the title.
- Finishing a month as #1 contender automatically schedules a full championship feud for the next month.
- If that championship feud occupies March, the annual #1 contender tournament moves to April.
