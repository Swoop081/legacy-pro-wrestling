# LEGACY Pro Wrestling 8.6.2 — Jett Decision Pool Repair

- Jett Valentine now uses only his assigned 15 illustrated decisions.
- Generic decisions and tag/finisher fallback options can no longer enter Jett's pool.
- Missing or invalid Jett phase pools now raise a development error instead of silently substituting an unrelated option.
- Jett's decision result screen now reuses the exact selected artwork.
- Result rewards animate into view and retain the existing match score, control and crowd effects.
- The YOUR CALL tab is layered above the panel border rather than being clipped by it.
- All other wrestlers continue using the existing decision framework.
